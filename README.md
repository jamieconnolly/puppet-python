# Python Puppet Module for Boxen

[![Build Status](https://travis-ci.org/boxen/puppet-python.png?branch=master)](https://travis-ci.org/boxen/puppet-python)

## Usage

```puppet
include python
```

## Required Puppet Modules

* `boxen`, >= 3.1
* `java`, >= 1.2
* `module-data`
* `repository`, >= 2.2
* `stdlib`, >= 4.0
